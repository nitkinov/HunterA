**HunterA** — это продвинутый фреймворк для пентеста, созданный специально для устройств Android без root-прав. Он работает в Termux и объединяет множество мощных инструментов: сканеры портов, разведку WHOIS/DNS, поиск уязвимостей (CVE), веб-фаззер, генератор паролей, сетевой сниффер и не только.

> ⚠️ **Только для образовательных целей и тестирования собственных систем. Автор не несёт ответственности за незаконное использование.**

## 🚀 Возможности

- **Широкий набор модулей** для пентеста и OSINT
- **Не требует root-прав** (использует TCP Connect, PCAPdroid и Python raw sockets)
- **Полностью асинхронный движок** для высокой производительности
- **Поддержка Nmap, tcpdump, tshark** и других консольных утилит
- **Интеграция с Termux:API** (Wi-Fi сканирование, уведомления)
- **Умная auto-detection** кодировок, хешей и типов данных
- **Шифрование конфиденциальных данных** (API-ключи, пароли)
- **Автоматические резервные копии** и восстановление конфигурации

## 📋 Категории модулей

- **Сканирование** – портов, хостов, подсетей (с граббингом баннеров и поиском CVE)
- **Разведка** – WHOIS, DNS, поддомены, технологии сайтов, OSINT по email/телефону/username
- **Анализ уязвимостей** – поиск CVE по продуктам и баннерам, сканер веб-уязвимостей (SQLi, XSS, LFI и др.)
- **Эксплуатация** – брутфорс, генерация словарей, проверка стойкости паролей
- **Сниффинг** – захват и анализ трафика (tcpdump, tshark, PCAPdroid)
- **Утилиты** – конвертер данных (40+ форматов), менеджер паролей, хеширование
- **Настройки** – гибкое управление конфигурацией, профилями и API-ключами
